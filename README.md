# GuessGame
Guessing Game
This is a simple number guessing game built with HTML, CSS, and JavaScript. The objective of the game is to guess the secret number, which is randomly generated between 1 and 20. The game provides feedback on each guess, indicating whether the guess is too high or too low. The player's score decreases with each incorrect guess, and the game ends when the player guesses the correct number or runs out of attempts.

Features
Random Number Generation: Each game generates a random number between 1 and 20.
User Input: Players can input their guess using a text input field.
Feedback: The game provides feedback on whether the guess is too high, too low, or correct.
Score Tracking: The player's score starts at 20 and decreases with each incorrect guess.
High Score Tracking: The game keeps track of the highest score achieved during the session.
Game Reset: Players can reset the game at any time to play again.

How to Play

1.Enter a number between 1 and 20 in the input field.
2.Click the "Check!" button to submit your guess.
3.Read the feedback message to see if your guess is too high, too low, or correct.
4.Keep guessing until you find the correct number or run out of points.
5.Click the "Again!" button to reset the game and play another round.
